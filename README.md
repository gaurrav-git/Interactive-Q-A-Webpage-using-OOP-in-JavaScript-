# Interactive Q&A Webpage

## Overview
This project is an interactive FAQ-style webpage where answers toggle visibility when the corresponding questions are clicked. It was built using HTML, CSS, and Object-Oriented JavaScript.

## Implementation Details
* **HTML:** Uses semantic structure to logically group question-answer pairs.
* **CSS:** Features hover states, visual cues (+/-), and a smooth `max-height` transition to animate the expanding and collapsing of answers.
* **JavaScript (OOP):** Implements a `QAItem` class to store question/answer data and encapsulate the rendering and toggling logic. An array of these objects is used to dynamically generate the DOM elements.
